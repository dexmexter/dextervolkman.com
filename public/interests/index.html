Interests

Computers

I’m completely self-taught and have been learning about Linux and computers since college. I would like to say that it is just a hobby but at this point it might be more on an addiction, now that I’ve peeked behind the curtain of computer technology, I’m not sure I could stop looking for more knowledge if I tried ¯\(ツ)/¯

My curiosity is likely the biggest reason I was attracted to the Linux operating system. The fact that I can dig down and read the source code for everything my system is doing just blows me away. I will never know everything about the computers I use but I am going to keep learning as much as I can.
